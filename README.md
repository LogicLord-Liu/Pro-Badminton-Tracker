## 🏸 Pro Badminton Tracker(PBT)

**一款专为羽毛球爱好者打造的专业计分与复盘工具。**

Pro Badminton Tracker 旨在提供最流畅、最直观的计分体验。采用 iOS 设计语言，支持丝滑的暗黑模式切换，并具备精准的比赛走势记录与图片导出功能。

- ## ✨ 项目特色

  - **完全开源免费**：没有任何内购、没有任何隐藏费用。
  - **纯净无广告**：回归于工具本质，拒绝干扰，专注于每一分球。
  - **IOS视觉风格**：基于 Cupertino 设计语言，极致简约，操作触手可及。
  - **全平台支持**：基于 Flutter 开发，支持 Android 与 iOS（即将支持更多平台）。

- ## 🚀 主要功能

  - **专业计分系统**
    - **灵活赛制**：支持 1 局、3 局 2 胜或 5 局 3 胜制。
    - **标准 BWF 逻辑**：严格遵循世界羽联计分规则（包括 11 分间歇提醒、21 分及加分规则）。
  - **深度历史回溯**
    - **每一分记录**：自动记录比赛中产生的所有得分点。
    - **长按回溯 (Revert)**：在历史记录面板中，长按任意比分点可瞬间将比赛状态恢复到该时刻，轻松修正误判。
    - **分局统计**：清晰展示每一局的比分走势。
  - **主题与交互**
    - **一键主题切换**：支持明亮/暗黑模式，适配各种光线环境。
    - **震感反馈**：计分操作伴随细腻的线性马达反馈，增强操作确认感。
  - **数据导出与分享**
    - **生成战报图片**：一键生成长图战报，包含对阵双方、最终比分及走势。
    - **系统级分享**：支持保存至系统相册或直接发送给好友，展示你的胜利时刻。

- ## 🛠️ 技术细节

  - **核心框架**: Flutter (Material 3 + Cupertino)
  - **状态管理**: `ChangeNotifier` + `Provider` 思想
  - **关键库**:
    - `gal`: 跨平台相册保存
    - `share_plus`: 系统原生分享
    - `render_repaint_boundary`: UI 转图片导出
